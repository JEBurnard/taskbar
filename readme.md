# Windhawk Mini
Minimal build of [Windhawk](https://github.com/ramensoftware/windhawk) that includes the following Windows customisations:
* [Disable taskbar grouping](https://github.com/ramensoftware/windhawk-mods/blob/main/mods/taskbar-grouping.wh.cpp)
* [Middle-click close](https://github.com/ramensoftware/windhawk-mods/blob/main/mods/taskbar-button-click.wh.cpp)

The purpose of this project is to allow me to customise my work desktop without being blocked by FortiClient.

Many thanks to [m417z](https://github.com/m417z) for all the inspration
* [WindHawk](https://windhawk.net/)
* [7 Taskbar Tweaker](https://ramensoftware.com/7-taskbar-tweaker)
